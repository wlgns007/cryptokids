Cold-start checklist (every time you come back)

Start the local chain

cd C:\Users\iBuyPower\contracts
npx hardhat --config .\hardhat.config.mjs node


In a second PowerShell window: set parent PK and deploy

cd C:\Users\iBuyPower\contracts
$env:PK="0x<parent private key from node window Account #0>"
node .\scripts\deploy_oya.mjs
Get-Content .\addresses.json


New chain = new contract addresses. Copy the fresh addresses into your HTML.

Open the app

Run a tiny server: python -m http.server 8080

Open http://127.0.0.1:8080/parents-shop.html
